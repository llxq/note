-> Git 远程
    -> 在 其他服务器上创建共享仓库
    -> 创建共享仓库 
        -> 首先创建一个以 .git结尾 的目录
        -> 必须是一个 裸仓库
            -> 裸仓库就是里面任何工作目录
            -> git init --bare (就是创建一个裸仓库)

    -> 将 master 共享至远程仓库
        -> git push 远程仓库地址 master
            -> git student@192.168.141.95(用户地址):需要存放的目录  分支名字

    -> 拿到远程仓库中 master中的版本
        -> git pull 远程仓库地址 master

    -> 克隆仓库
        -> git clone 用户名@ip地址:需要克隆到的地址 文件夹名字
            -> 克隆会把仓库也给克隆下来
