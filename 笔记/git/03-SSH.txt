-> SSH
    -> SSH 是一种网络协议，用于计算机之间的加密登录
    -> SSH 只是一种协议，存在多种实现，即有商业实现，也有开源实现。这里针对的是 OpenSSH，它是自由软件，应用非常广泛
    -> 如果要在 window系统中使用 SSH，会用到另一种软件 PuTTY，我们后面用到的 Git 客户也集成了 SSH
    -> 格式：ssh user@host
        -> user 代表真实存在的用户 host 代表
    -> SSH属于非对称协议

-> 加密方式
   -> 对称性加密：就是加密与解密使用的是同一种方式
   -> 非对称性加密：加密与解密使用的不是同一种方式 

-> 通过 ssh登录
    -> 命令行输入：ssh 用户名@ip地址
    -> 密码：是创建的用户的密码
