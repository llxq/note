-> Git分支
    -> 在我们现实开发中，需求而往往是五花八门的，同时开发多个需求的情况十分常见，比如当你正在专注开发一个功能时，突然有一个紧急的 BUG 需要你来修复，这个时候我们当然是希望在能够保存当前任务进度，再去修改这个BUG，等这个BUG修复完成之后再继续我们的任务。
        -> 这个时候我们就可以通过 Git 创建分支来解决开发中类似的问题
    
    -> 创建分支
        -> git branch 分支名称

    -> 查看分支
        -> git branch
        -> * 在那个位置，表示我们当前处于哪个分支

    -> git checkout 分支名：切换到某一个分支

    -> 合并分支
        -> git merge 分支名称
            -> 可以将两个分支中的内容进行合并

    -> 删除分支
        -> git branch -d 分支名字
            -> 如果有未保存的文件，是不能删除该分支的
        -> git branch -D 分支名字  
            -> 强制删除该分支

    -> git remote add 别名 用户名@ip地址:路径
        -> 就是给后面的 用户名@IP地址 起了一个别名
        -> git remote add itcast student@192.168.141.95:/home/student/itcast.git
    -> 后面使用该命令的时候可以这样写了
        -> git pull 别名
            -> git pull itcast

    -> git remote 
        -> 查看别名
        -> 可以查看已添加的远程主机

    -> git remote '别名'
        -> 查看当前别名的真实地址
