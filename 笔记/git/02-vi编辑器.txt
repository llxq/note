-> vi编辑器
    -> 如同 window 下的记事本，vi编辑器 是 Linux 下的标配，通过它我们可以创建、编辑文件。它是一个随系统一起安装的文本编辑软件

-> vi编辑器 的三种模式
    -> vi编辑器提供了 三种模式，分别是：命令行模式、输入模式、底行模式，每种模式下用户所能进行的操作是不一样的
    -> 首先我们打开 vi编辑器的时候，我们是处于命令行模式，在处于命令行模式的时候，我们是不能对文件进行编辑的
    -> 从其他模式切换到命令行模式：按 Esc键
    -> 从命令行模式切换到输入模式：按 i键 或者 a键
    -> 从命令行模式切换到底行模式：按 shift + : ，就是输入一个 英文状态下的 :
        -> 保存： :w
        -> 退出： :q
        -> 保存并退出： :wq
        -> 强制退出： :q!   (没有保存是不能退出的)

-> vi编辑器三种模式中的操作
    -> 底行模式：
        -> :w   保存，不会退出
        -> :w 需要保存的路径    另存
        -> :q   退出
        -> :wq  保存并退出
        -> :q!  不保存，并且强制退出
        -> :e!  撤销更改，返回到上一次保存状态
        -> :set nu  设置行号，就是在边上加了一行行号
    -> 命令行模式
        -> ZZ   保存并退出
        -> u    撤销操作，可使用多次
        -> dd   删除当前行 
        -> yy   复制当前行
        -> p    黏贴
        -> Ctrl + f     向上翻页
        -> Ctrl + b     向下翻页
        -> i    进入编辑模式，当前光标处插入
        -> a    进入编辑模式，当前光标后一行插入
        -> A    进入编辑模式，当前行尾插入
        -> O    进入编辑模式，当前行上面插入新行，自动在上面开了一个新行，光标在上面
        -> o    进入编辑模式，当前行下面插入新行，自动在下面开了一个新行
    -> 当我们处于编辑模式的时候

-> vi 文件名：编辑文件
    -> i ：编辑，按下 i 键的时候就可以编辑其中的内容了
    -> 如果想要退出 vi 编辑器，先按下 Esc键，然后输入 :q! 即可强制退出
